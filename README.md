# Cybernetic-AR-HUD

#An open-source, stylized Augmented Reality Heads-Up Display that brings the cyberpunk future to your world. This project overlays a dynamic, interactive interface onto your real environment through AR devices, displaying vital information as if it were part of your vision.

✨ Features
🖥️ Immersive Cybernetic Aesthetics: Glowing lines, data panels, and sci-fi typography for authentic cyberpunk feel

📊 Real-time System Monitoring: Live display of device CPU, RAM, battery, and network status in AR

🔔 Dynamic Notification System: Smooth animated alerts for messages, emails, and system events

🧩 Modular Widget Framework: Easily customizable UI components with drag-and-drop functionality

🌐 Environmental Integration: AR Foundation powered spatial awareness and environment tracking

🎨 Custom Shaders & VFX: Advanced visual effects using Unity's Shader Graph and VFX Graph
